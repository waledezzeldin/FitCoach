## Appendix A: Image Optimization Guidelines

### File Size Targets
- **Icons/Logos**: < 50KB
- **Thumbnails**: < 100KB
- **Product Images**: < 200KB
- **Hero Images**: < 500KB
- **Background Images**: < 300KB

### Format Recommendations
- **Logos/Icons**: SVG (scalable) or PNG with transparency
- **Photographs**: JPG (lossy compression acceptable)
- **Illustrations**: SVG or PNG
- **Animations**: Lottie JSON or GIF (< 1MB)

### Responsive Image Sizes
For each image, provide 3 sizes:
1. **Mobile**: 640px width
2. **Tablet**: 1024px width
3. **Desktop**: 1920px width

### Naming Convention
```
{category}_{description}_{size}.{ext}

Examples:
- workout_pushup_800x600.jpg
- icon_home_64x64.svg
- logo_primary_light_512x512.png
- product_protein_whey_600x600.jpg
```
