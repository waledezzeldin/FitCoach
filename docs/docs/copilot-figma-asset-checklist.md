## Export Instructions
- Export all icons as SVG (preferred) or PNG at 2x/3x resolution for mobile.
- Organize assets in `mobile/assets/` by screen or category.
- Name files clearly (e.g., `icon_gender_male.svg`, `hero_intro_workout.png`).
- Document any missing or ambiguous assets for clarification.
